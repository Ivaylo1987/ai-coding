# The Practical CCA Prep Guide

A practical, action-oriented study guide for the Claude Certified Architect (CCA) Foundations exam. Courses to complete, docs to read, projects to build, patterns to drill, and checklists to track — all mapped to the 5 exam domains.

## [View the live guide →](https://ivaylo1987.github.io/hands-on-ai/guides/cca/)

## Exam domains

| Domain | Weight | Focus |
|--------|--------|-------|
| D1 — Agentic | 27% | Agentic loop, multi-agent patterns, hooks, session management |
| D2 — Tools | 18% | Tool design, MCP servers, function calling |
| D3 — Claude Code | 20% | Claude Code workflows, team configuration |
| D4 — Prompts | 20% | Prompt engineering, structured data extraction |
| D5 — Reliability | 15% | Evaluation, guardrails, error handling |

## What's inside

- Domain breakdowns with specific resources and study paths
- 5 hands-on projects covering all 6 exam scenarios
- Checklists to track progress across domains
- Links to official Anthropic docs, blogs, and courses
